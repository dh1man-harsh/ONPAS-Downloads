# ONPAS Downloads

This repository contains only public ONPAS installers, release metadata, and compiled release artifacts.

## Windows

```powershell
irm https://raw.githubusercontent.com/dh1man-harsh/ONPAS-Downloads/main/install.ps1 | iex
```

## macOS

```bash
curl -fsSL https://raw.githubusercontent.com/dh1man-harsh/ONPAS-Downloads/main/install.sh | bash
```

## Latest Version

`latest.json` points installers to the newest ONPAS release assets.
